Este botão é conhecido como D-Pad, que significa "Directional Pad" (Painel Direcional).

É composto por quatro direções (cima, baixo, esquerda e direita) e é usado para movimentar Mario e interagir com o jogo.

Direcional para Cima (↑): Move Mario para cima/ Para Baixo (↓): Faz Mario se agachar ou descer.

Para Esquerda (←): Move Mario para a esquerda/ Para Direita (→): Move Mario para a direita.
